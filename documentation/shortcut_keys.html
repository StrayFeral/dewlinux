Common shortcut keys for some applications

VIM

  • 0 – Disable spellcheck (default mode)

  • 1 – Enable spellcheck for English

  • 2 – Enable spellcheck for Bulgarian

  • 3 – Enable spellcheck for Russian

  • 4 – Enable spellcheck for French

  • [s ]s – Previous/next misspelled word

  • z= - Invoke suggestions for the currently selected spellcheck language

  • x X – ALE PLUGIN – Previous/next issue

NEOMUTT

  • , . – Previous/next mail folder

  • m r f – New Mail, Reply, Forward

  • v – View attachments
