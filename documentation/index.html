WHAT IS INSTALLED AND WHY

Since this experiment assumes a system for personal home use, the folowing use cases were taken into consideration with the folowing utilities installed:

  • A bootable and operational computer as a base
    • Debian linux
  • Basic accessibility option - manual font setup
    • dpkg-reconfigure console-setup
  • Additional keyboard layout setup and switching (left ALT + left SHIFT to switch)
  • Basic system management and system information
    • manual system management
    • fastfetch - system info
    • htop, btop - process monitoring
    • eza - ls replacement (I already aliased ls with it)
    • fzf - fuzzy finder
    • ncdu - du replacement
    • bat - cat replacement
    • dysk, duf - df replacements
    • rg - grep replacement (ripgrep)
    • fd - find replacement (fdfind)
  • Comfortable shell with some coloring
    • bash
  • Desktop organization
    • tmux
  • Comfortable file management
    • locate
    • mc
  • Text editing - text files
