IRC HELP (WEECHAT)

IRC stands for Internet Relay Chat. It is a very old way to talk to other people.

IRC General Information

If you have zero idea what IRC is and looks like, check the following links in the end of this section.

In short - in the IRC chat you type commands. Each command starts with the slash symbol "/" and continues with a word. There are global commands which could be executed in any chat window, like "/connect", "/disconnect", "/join", "/part", "/leave", "/msg", "/nick" and more.

How to setup Libera.Chat (INITIAL SETUP)

You need to do what's described in this section only once.

You need to type the following commands first (beware not to copy the list bullets!):

(and be sure to replace "YOURNICKNAMEHERE" with a nickname of your choice - put in any case you want), however using real names is not advisable!

  • /server add libera irc.libera.chat/6697 -tls
  • /set irc.server.libera.nicks "YOURNICKNAMEHERE,YOURNICKNAMEHERE2,YOURNICKNAMEHERE3,YOURNICKNAMEHERE4,YOURNICKNAMEHERE5"
  • /set irc.server.libera.username "YOURUSERNAMEHERE"
  • /set irc.server.libera.realname "YOURREALNAMEHERE"

When you setup default nicks for a server you can set any number of nicks. The idea is - upon connection if the first nickname already exists, it means "it's taken" and you need to quickly set new nick for 1 minute. When you set multiple nicks this would be done automatically - WeeChat will automatically set the second nick, if it's taken too it will set the third nick etc.

How to chat in WeeChat

From now on what you need to do is this - launch `weechat` and then it will await commands. Here are the most useful ones:

  • /connect libera
  • /disconnect
  • /quit
  • /nick YOURNEWNICKHERE
  • /me SOMETHINGHERE
  • /join CHANNELNAMEHERE
  • /part
  • /leave
  • /msg TARGETNICKNAMEHERE private message text goes here

While some are obvious on what they do, here is a brief explanation:

  • /connect libera - connects to Libera.Chat
  • /disconnect - disconnects from the currently connected server
  • /quit - disconnects and quit WeeChat
  • /nick YOURNEWNICKHERE - changing your nickname mid session
  • /me SOMETHINGHERE - saying something in public and accenting it
  • /join CHANNELNAMEHERE - joining a chat channel
  • /part - leaving the current chat channel
  • /leave - same as above
  • /msg TARGETNICKNAMEHERE private message text goes here - sending a private message to a user

Examples:

  • /connect libera
  • /join #mytestchannel1
  • /nick im-so-cool
  • /me is going insane
  • /msg john321 hey did you noticed Jane is gone?
  • /quit

NOTE: "/quit" usually allows a message, which is the quit reason, so you could type "/quit must go for coffee"

NOTE: Nicknames and channel names accept both dashes and underscores in the name, as well as numbers in the end. Better avoid special characters.

Important: If you mistype the name of a channel or you just type the name of a channel which does not exists, you will be joined to a brand new channel and you will automatically become the ChanOp of this channel.

Regular users and operators: Basically there are few roles for users - regular users, channel operators (ChanOp), server admins and I think there were few other roles I can't recall. If you join an existing channel you would be a regular user. However if you want to create your own channel, just join a channel with non-existing name and it will be your channel and you will me automatically assigned the role of a ChanOp.
